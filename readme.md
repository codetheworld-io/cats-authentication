# Cats-Authentication

## Develop

### Start mongo service with Docker

```shell
docker run -it --rm -p 27017:27017 mongo
```

### Install dependencies

```shell
npm ci
```

### Run test

Run all tests

```shell
npm run test
```

Run test with watching feature

```shell
npm run watch -- --watch
```


### Start dev server

```shell
npm run dev
```
